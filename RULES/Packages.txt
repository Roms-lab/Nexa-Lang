Nexa Package Rules (nexapkg)
============================

INSTALL
  nexapkg install <pkg>              Add + install (nexapkg add <pkg> && nexapkg install)
  nexapkg install --global <pkg>     Install to ~/.nexa/packages (system-wide)
  nexapkg add <pkg> [dir]           Add to nexapkg.json
  nexapkg add as <path> <pkg>       Add with custom include path
  nexapkg add <pkg> as <path>       Same
  nexapkg install [dir]              Install from nexapkg.json

INCLUDE SYNTAX
  #include <pkg/Module>              Include package module
  #include <Win32/Audio>             Custom path from "Win32/Audio" in manifest
  #include <pkg/Win32>               Submodule within package

  NO AUTO INCLUDES - all includes must be explicit.

CUSTOM PATHS (Rule.txt)
  nexapkg add as Win32/Audio ./libs/Audio
  -> nexapkg.json: "Win32/Audio": "file:./libs/Audio"
  -> #include <Win32/Audio>

  nexapkg add as Win32 ./packages/Win32
  -> "Win32": "file:./packages/Win32"
  -> #include <Win32/MessageBox> or <Win32/Module>

MANIFEST (nexapkg.json)
  {
    "name": "myapp",
    "dependencies": {
      "greet": "file:./packages/greet",
      "Win32/Audio": "file:./libs/Audio.nxa",
      "Win32": "file:./packages/Win32"
    }
  }

  Key = include path (#include <Key>)
  Value = source (file:path or https://github.com/user/repo)

PACKAGE STRUCTURE
  - "greet": "file:./greet" -> packages/greet/greet.nxa (folder with greet.nxa)
  - "Win32/Audio": "file:./Audio.nxa" -> packages/Win32/Audio.nxa (single file)
  - "Win32": "file:./Win32" -> packages/Win32/* (folder contents)
